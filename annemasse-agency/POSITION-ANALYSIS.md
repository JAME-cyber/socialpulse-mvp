# SocialPulse Annemasse — Position Divergente: Analyse Honnête

## ⚡ Réponse courte: Non, on n'est PAS parfaitement positionné.

Le POC est un excellent **outil de prospection**, mais le positionnement "site vitrine 350€ one-shot" a 5 failles structurelles.

---

## 5 Failles de la Position Actuelle

### 1. Produit Commodity
Un site vitrine = produit de commodité. Wix: 15€/mois. WordPress.com: 4€/mois.
On est dans la zone morte: "too expensive for DIY, too cheap for custom".

### 2. Marché Trop Petit
318 PME réellement adressables sur l'agglo. Après ça, il faut étendre géographiquement.

### 3. Pas de Récurrent
One-shot 350€. Pas d'ABO. Pas de LTV. Pas de business durable.

### 4. Le Site Vitrine Meurt
Google My Business + Instagram remplacent le site pour les recherches locales.
Un restaurant à Annemasse: 90% du traffic vient de Google Maps, pas d'un site web.

### 5. Proximité Fake
On dit "basé à Annemasse" mais on est un script Python. Le client veut un humain.

---

## 3 Positions Divergentes Qui Marchent

### Position A: "Google Maps Booster"
- **On vend pas un site. On vend des clients via Google Maps.**
- Produit: Optimisation GMB + fiche Maps pro + site 1 page
- Prix: 297€ setup + 49€/mois
- TAM: 2 382 PME × 49€/mois × 5% = 5 825€ MRR
- Edge: On a DÉJÀ les données Maps

### Position B: "White Label Agency-in-a-Box"
- **On vend pas aux PME. On vend l'outil aux agences.**
- Produit: Pipeline SocialPulse en SaaS
- Prix: 97€/mois/agence
- TAM: 200 agences Rhône-Alpes × 97€ = 19 400€ MRR
- Edge: Le POC = le MVP du SaaS

### Position C: "Photographe+ Digital Local"
- **Le RDV physique EST le produit. Le site est l'upsell.**
- Produit: Photos pro gratuites → upsell site + gestion Instagram
- Prix: Photos gratuit → site 350€ → IG 99€/mois
- TAM: 1 641 sans site × 10% conversion × 99€/mois
- Edge: Impossible de refuser du gratuit. Trust physique.

---

## Verdict: Position Gagnante = A + C Hybride

```
Phase 1 (S1-4):  Photos gratuites → site 350€ → IG 99€/mois
Phase 2 (M2-3):  Google Maps Booster 49€/mois
Phase 3 (M4+):   White label SaaS 97€/mois/agence
```

### Le Rôle du POC

Le POC n'est PAS le produit. Le POC est l'**outil** qui alimente les 3 phases:
- 2 352 leads = base de prospection
- Scoring = priorisation des RDV photos
- Templates = livraison rapide
- Vidéos HyperFrame = contenu Instagram géré
- Cold messages = prise de RDV
- Journal WORM = conformité

**Le client achète un RÉSULTAT (plus de clients), pas un site.**
